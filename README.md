# Nino_Salvaje
 
Parche programado en PD para ejecutar Niño Salvaje.

El Archivo nebula es un shader que corre sobre GEM. 

El archivo texto es la qlist que se ejecuta cada vez que se presiona la tecla espaciadora en PD.

El archivo BS_Texto contiene indacaciones para los controlar las variables en el texto.
